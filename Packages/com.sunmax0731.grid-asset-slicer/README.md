# Unity Grid Asset Slicer Package

Unity Grid Asset Slicer is a Unity Editor extension package for slicing a grid-based source image into individual PNG assets.

## Status

This package is currently in the `0.1.0` foundation phase. The package scaffold, assembly definitions, and release planning are in place. Functional slicing services and Editor UI are tracked in GitHub Issues.

## Package Name

```text
com.sunmax0731.grid-asset-slicer
```

## Public Menu Path

Planned menu path:

```text
Tools > Grid Asset Slicer
```

## Development Order

1. Core grid model and calculator
2. Slice session JSON persistence
3. Export naming and conflict behavior
4. PNG cell extraction and export
5. Editor window workflow
6. Validation and release packaging

See the repository-level `docs/` folder for requirements, design, validation, and release planning.
