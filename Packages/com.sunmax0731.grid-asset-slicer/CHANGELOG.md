# Changelog

## 0.1.2 - 2026-05-04

- added a larger draggable margin controller and slider-assisted grid editing while keeping numeric entry
- added per-column and per-row variable boundary editing with session persistence
- allowed uneven implicit grid sizes to preview and export using the same calculated rectangles
- added optional uniform export resize width and height for all written PNG files
- refined the main IMGUI layout with foldout settings groups and clearer workspace states

## 0.1.1 - 2026-04-26

- unified the Unity Editor menu structure to `メイン画面`, `ライセンス`, and `バージョン情報`
- added the license/version information window
- documented MIT License in the repository README, package README, and release docs
- refreshed manuals, validation notes, and release copy for the current implementation


All notable changes to Unity Grid Asset Slicer will be documented in this file.

## 0.1.0

- Added initial UPM package scaffold.
- Added Runtime and Editor assembly definitions.
- Established package identity for `com.sunmax0731.grid-asset-slicer`.
- Added grid calculation, session JSON persistence, export naming, PNG export, and Editor window workflow.
- Added deterministic sample PNGs and replay fixtures.
- Added validation gate and package documentation for BOOTH release preparation.
